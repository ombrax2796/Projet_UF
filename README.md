# Projet_UF
## Cahier des charges

Dans ce projet nous créerons au moins un jeu de cartes ou au mieux une application qui permet d'accéder à plusieurs jeux de cartes tels que "le Président", le "Dutch" ou bien le "Poker".

### A - Fiche produit

Moteur : Maya ou ZBrush
